# Pars Channel by @MaxGolang

GUI приложение для парсинга участников Telegram каналов на Golang с использованием Fyne.

## Установка

```bash
go mod download
```

## Запуск

```bash
go run main.go
```

## Функции

- **Парсер каналов Telegram** - парсинг участников каналов в реальном времени
- **Таблица пользователей** - отображение спарсенных пользователей с информацией (ID, имя, username, premium, bot, restricted)
- **Экспорт данных** - скачивание результатов в различных форматах:
  - CSV (полная информация)
  - TXT с usernames
  - TXT с IDs
  - Все файлы сразу
- **Статистика** - отслеживание прогресса парсинга в реальном времени

## Использование

1. Запустите программу: `./pars` или `go run main.go`
2. Перейдите на вкладку "Парсер"
3. Введите данные:
   - API ID (из https://my.telegram.org)
   - API Hash
   - Bot Token (от @BotFather)
   - Channel ID (например: -1001234567890)
4. Нажмите "Начать парсинг"
5. Наблюдайте за результатами в таблице в реальном времени
6. После завершения используйте кнопку "Скачать" для экспорта данных

## Зависимости

- fyne.io/fyne/v2 - GUI фреймворк
- github.com/gotd/td - Telegram API клиент
